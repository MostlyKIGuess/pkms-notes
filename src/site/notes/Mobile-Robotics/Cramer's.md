# Cramer's Rule
$$Ax = b$$
Where A is a square matrix of coefficients, x is the vector of unknowns, and b is the result vector.
Cramer's Rule solution:
$$x_{i} = \frac{det(A_{i})}{det(A)}$$
Where $A_{i}$ is the matrix formed by replacing the i-th column of A with vector b.
